# biz_archive_for_ima
公众号文章知识库所有快照-这里没有就是真没有了-**为什么你不搞私人仓库-继续看你会明白的**

这是一个什么项目：？bushi名字好诡异，本意是为了存储公众号文章防止失效，我尝试了bd,qk网盘分享都不支持在线预览，使用123云盘因为我整理了很多原作者分享附件**体积很大**，而且分类是按照作者而不是这里的时间分类

一些小众云盘也没有很好的效果，这里**特别感谢github**提供的在线预览功能，通向知的道路不应该被链接过期，琐碎找不到迷失方向，原文地址可以在**html文件头**找到【都是edge-captured的】，当然也可以按照公众号名字转到wx搜索

此外，作为经常找教程看新闻的科技爱好者，很建议构架一个ima知识库喂给不错的deepseek（本地模型），让AI更懂好用就要先提供个人优质信息嘛，至于工作流可以直接看一下这篇文章
